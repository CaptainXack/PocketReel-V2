# Get the APK onto your phone, then onto your Fire Stick

Once you have a built APK, you do not have to use a PC every time.

## Option 1: Download the APK to your phone

- Download the APK artifact from GitHub Actions on your phone.
- Keep it somewhere easy to find, such as Downloads.

## Option 2: Move the APK to the Fire Stick

Use any method you already trust:

- a file transfer app on both devices
- a cloud drive you can open on the Fire Stick
- a local network file share
- a web server you control
- Downloader on the Fire Stick with a direct link to the APK

## Final install step on Fire Stick

When the APK is on the Fire Stick:

1. Open the file from your file manager or transfer app.
2. If asked, allow installs from that app.
3. Confirm the install.
4. Open **CX Fire Loader** from the Apps screen.

## First in-app test

For the easiest first test:

1. Switch to **Direct URL** mode.
2. Paste a direct HTTPS APK link that you control.
3. Start the download.
4. Let CX Fire Loader hand the APK to the Android installer.
